Morning surf sesh.
- Take off.
- Bottom turn.
- Cut back.
- Pump.

- Spiderman stretches.
- Monkey jumps.

- Push ups.
- Core training.
- Pull ups.
- Legs.

Freediving (once a month).

BJJ (Wed Fri)
